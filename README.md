# 🥮 烤肉準備項目工作區 🔥

## 📖 項目簡介

烤肉準備項目工作區是一個專為家庭烤肉聚會設計的多人協同管理系統。透過整合 Google Sheets 作為後端數據庫，實現了真正的即時協同編輯功能，讓所有家庭成員都能參與準備工作的管理。

## ✨ 主要特色

### 🤝 多人即時協同
- **Google Sheets 整合**：以 Google Sheets 作為數據後端
- **即時同步**：所有成員的操作立即同步到雲端
- **跨設備支援**：手機、平板、電腦都能使用
- **離線友好**：支援 Google Sheets 離線編輯功能

### 📱 現代化介面
- **響應式設計**：自適應各種螢幕尺寸
- **直觀操作**：簡潔易用的使用者介面
- **即時反饋**：操作狀態即時提示
- **視覺回饋**：優雅的動畫和過渡效果

### 🛠️ 完整功能
- **項目管理**：新增、編輯、刪除準備項目
- **狀態追蹤**：標記完成/未完成狀態
- **分組篩選**：按家庭分組或狀態篩選
- **數據持久化**：所有變更自動保存到雲端

## 🏗️ 系統架構

```
前端網頁應用 ←→ Google Apps Script API ←→ Google Sheets 數據庫
     ↕                    ↕                      ↕
用戶界面交互          API 路由處理           數據存儲與同步
```

## ⚙️ 技術實現

### 前端技術
- **HTML5 + CSS3**：現代化網頁結構和樣式
- **原生 JavaScript**：無框架依賴，載入快速
- **Fetch API**：與後端 API 通信
- **響應式設計**：CSS Grid + Flexbox 佈局

### 後端技術
- **Google Apps Script**：無服務器後端解決方案
- **Google Sheets API**：數據存儲和管理
- **RESTful API**：標準化的 API 設計
- **CORS 支援**：跨域資源共享

### 數據流程
1. 用戶在網頁上操作
2. 前端調用 Google Apps Script API
3. Apps Script 操作 Google Sheets
4. 數據變更即時同步到所有用戶

## 🎯 使用場景

### 👨‍👩‍👧‍👦 家庭聚會準備
- 分配各家庭的準備責任

### 📱 多設備協同
- 在超市用手機標記已購買項目
- 在家用電腦查看整體準備進度
- 平板上新增臨時想到的項目

### 🔄 即時同步
- A 在手機上標記完成 → B 的電腦立即更新
- 新增項目後所有人立即收到
- 避免信息不同步的問題

## 📊 數據結構

### Google Sheets 表格結構
| ID | 家庭分組 | 項目名稱 | 數量 | 狀態 |
|----|----------|----------|------|------|
| 1  | 郭家     | 草蝦     | 12尾 | incomplete |
| 2  | 哥家     | 牛小排   | 3盒  | completed |
| 3  | 翁家     | 啤酒     | 2打  | incomplete |

### API 數據格式
```javascript
{
  "success": true,
  "data": [
    {
      "id": 1,
      "groupName": "郭家",
      "itemName": "草蝦",
      "quantity": "12尾",
      "status": "incomplete"
    }
  ],
  "timestamp": "2025-09-19T11:16:51.000Z"
}
```

## 🔧 自定義配置

### 修改家庭分組
編輯 `index.html` 和 `google-apps-script.js` 中的分組選項：
```html
<option value="你的家庭">你的家庭</option>
```

### 新增項目類型
在 Google Sheets 中直接新增或修改項目，系統會自動同步。

### 自定義樣式
編輯 `styles.css` 來修改視覺樣式：
```css
:root {
  --primary-color: #FF6B35;  /* 主色調 */
  --secondary-color: #F7931E; /* 輔助色 */
}
```
